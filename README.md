# Игра «Asteroids»
    
## Установка 
Написать в консоле:

    pip install pygame
    

## Запуск
Написать в консоле:

    python start.py 
    ввести имя

## Управление 
    → и ← - поворот 
    space - ускорение
    ↑ - выстрел/ предыдущий пункт меню
    ↓ - следующий пункт меню
    shift - выбор пункта меню
